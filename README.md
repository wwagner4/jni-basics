# jni-basics
Basic usage of JNI

## Features
* Simple HelloWorld. No arguments no return
* Return a basic type
* Basic type argument
* String argument
* Array of basic types argument
* Object as argument with access of the following members: Basic type, array of basic types, String
* Access members of an object array
